Important FAQ

What is the document model(DOM)?

The Document Object Model (DOM) is a programming API for HTML and XML documents. It defines the logical structure of documents and the way a document is accessed and manipulated.

How do you select an element from the DOM?

The selectors API provides methods that quick and easy to retrieve element nodes from the DOM by matching against a set of selectors.

What id event deligation in the context of the Dom, and why is it usefull?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an elements attributes and styles using the DOM?

The classlist object only provide methods to change the class of an element. If you want to change other attributes like id, href, or src can use the setAttribute() method.
